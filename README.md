# RVA Hacks 2026 — Central Ideas Hub

**Richmond Civic Hackathon · March 27–29, 2026 · VCU Snead Hall**

This repository is the one-stop shop for exploring and refining ideas from RVA Hacks 2026. Teams built prototypes addressing real problems facing Richmond, organized around the Mayor's Action Plan (MAP) seven civic pillars.

---

## Explore Ideas

Browse `ideas/<pillar>/` to see what was built and what's being developed further.

Each pillar directory contains:
- **`NEW_IDEAS.md`** — Top-scoring solution ideas generated from research and gap analysis
- **`NEW_IDEAS_RUBRIC_GRADES.md`** — Desirability rubric scores (Impact, Feasibility, User Value, Execution, Innovation, Equity)
- **`NEW_IDEAS_BLUESKY.md`** — Additional ideas generated via Blue Ocean ERRC reframing
- **`NEW_IDEAS_BLUESKY_RUBRIC_GRADES.md`** — Desirability scores for ERRC ideas
- **`IMPLEMENTATION_REALITY_REVIEW.md`** — Survivability scores for ALL ideas in that pillar (demos + NEW + BLUESKY)
- **Per-project directories** — Lifecycle reviews, gap analyses, demo transcripts, video recordings, and research

## Scoring & Awards

**Desirability** — See [RUBRIC.md](RUBRIC.md) for the judging rubric. Projects are scored 1–5 on six categories (Impact, Feasibility, User Value, Execution, Innovation, Equity & Inclusion) and weighted differently for three awards:
- **Mayor's Choice** — Most implementable project
- **Moonshot** — Most visionary project
- **Pillar Awards** (7) — Best project within each pillar

**Survivability** — See [`ideas/IMPLEMENTATION_REALITY_RUBRIC.md`](ideas/IMPLEMENTATION_REALITY_RUBRIC.md) for the post-event prioritization rubric. Ideas are scored 1–5 on five dimensions (Ownership Clarity, Data Readiness, Tech Stack Fit, Integration Surface, Maintenance Burden) to determine what can actually ship.

**Lifecycle Completeness** — See [LIFECYCLE_RUBRIC.md](LIFECYCLE_RUBRIC.md) for the 10-stage innovation lifecycle used to evaluate whether demos solved their stated problems.
