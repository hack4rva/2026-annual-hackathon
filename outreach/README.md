# SME Outreach

This directory contains outreach materials for subject matter experts — city staff, nonprofit professionals, and community organizations — whose knowledge is required to move demos from prototypes to real solutions.

## Structure

```
outreach/
  emails/                              # Short email drafts — copy-paste into your email client
    city-storytelling/
    thriving-built-environment/
    thriving-city-hall/
    thriving-economy/
    thriving-families/
    thriving-inclusive-communities/
    thriving-neighborhoods/
  questions/                           # Full research + detailed questions — linked from emails
    city-storytelling/
    thriving-built-environment/
    thriving-city-hall/
    thriving-economy/
    thriving-families/
    thriving-inclusive-communities/
    thriving-neighborhoods/
```

**`emails/`** — Concise drafts (~20 lines) with a link to the full research and questions page. These are what you actually send.

**`questions/`** — Detailed reference pages with confirmed research findings, specific questions, prototype recommendations, SME context, and internal tracking. These are what the recipient clicks through to read.

## How To Use

1. Open the email file for the relevant contact in `emails/`
2. Fill in the `[CONTACT NEEDED]` fields
3. Personalize the intro sentence to match your relationship to the contact
4. Verify the questions page link resolves correctly on GitHub
5. Send via email; log the date in the email file's "Follow-Up Status" section
6. When a response is received, summarize it in the questions file and update `gap_analysis.md`
